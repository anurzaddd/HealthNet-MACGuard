# 🏥 HealthNet-MACGuard

**اولین ابزار متن‌باز ممیزی رمزهای پیش‌فرض تجهیزات پزشکی مبتنی بر MAC Address**

## چرا این پروژه؟
در حملات سایبری به بیمارستان‌ها (مثل حملات باج‌افزاری)، ۷۰٪ نفوذها از طریق دستگاه‌های پزشکی که **رمزهای پیش‌فرض کارخانه** را دارند، انجام می‌شود. 
ابزارهای تجاری مثل `Armis` یا `Medigate` این کار را انجام می‌دهند اما بسیار گران‌قیمت هستند. این اسکریپت یک **جایگزین رایگان و سریع** برای برآورد آسیب‌پذیری شبکه‌ی بیمارستان شماست.

## ویژگی‌های منحصربه‌فرد
- 🔍 اسکن بدون نیاز به نصب خاص (فقط پایتون ۳)
- 🩺 شناسایی خودکار برندهای معروف (GE, Siemens, Philips, Medtronic, ...)
- 🔑 نمایش رمزهای پیش‌فرض کارخانه‌ای که باید فوراً تغییر کنند
- 📄 تولید گزارش HTML برای ارائه به مدیران ارشد و ممیزی HIPAA
